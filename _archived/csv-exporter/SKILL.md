# CSV Exporter

Convert JSON data arrays into downloadable CSV format.

## Usage

Paste or provide JSON data and get a CSV string back with download capability.

## Examples

- "Export this data to CSV"
- "Convert the leads to a spreadsheet"
- "Download results as CSV"
