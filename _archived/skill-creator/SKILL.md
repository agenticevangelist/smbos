# Skill Creator

Generate new skills with the proper directory structure, SKILL.md, ui.json, and execute.ts.

## Usage

Provide a skill ID (slug), name, description, and define the inputs and output type.

## Examples

- "Create a new skill called email-sender"
- "Add a skill for tracking social media"
