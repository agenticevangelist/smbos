# Cron Scheduler

Schedule any SMBOS skill to run on a recurring cron basis.

## Usage

- Create a new schedule with a cron expression
- List existing schedules
- Delete or toggle schedules

## Cron Examples

- `0 9 * * 1-5` — Every weekday at 9 AM
- `0 */2 * * *` — Every 2 hours
- `0 0 * * 0` — Every Sunday at midnight
