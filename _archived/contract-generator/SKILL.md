# Contract Generator

AI-powered contract generation skill. Extracts structured fields from free-form text input using Claude, fills a contract template, and generates PDF/Markdown output.

## Features
- Extract 20+ contract fields from text using Claude AI
- Preview extracted fields before generating
- Generate PDF or Markdown contracts
- Support for GEL, USD, EUR, RUB currencies
- Template-based generation with variable substitution

## Usage
1. Paste contract text or agreement details
2. Preview extracted fields
3. Generate final contract document
