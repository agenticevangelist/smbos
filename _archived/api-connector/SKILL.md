# API Connector

Connect to any REST API and view the response.

## Usage

Provide a URL and optionally set the HTTP method, headers, and request body.

## Examples

- "Call https://api.github.com/users/octocat"
- "POST to https://httpbin.org/post with JSON body"
- "Test my API at https://example.com/api/health"
