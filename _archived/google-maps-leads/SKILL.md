# Google Maps Leads Skill

This skill allows searching for businesses on Google Maps and extracting detailed information including:
- Name
- Address
- Rating and Review Count
- Website and Instagram presence
- Phone Number
- Category

## Usage

Provide a search query and a location.

Example: "Find clothing stores in Yerevan"
- Query: clothing stores
- Location: Yerevan
