# SQLite Manager

Manage local SQLite databases with full SQL query support.

## Usage

- Run any SQL query (SELECT, INSERT, UPDATE, DELETE, CREATE TABLE)
- List all tables in a database
- Describe table structure
- Default database: `smbos.db`

## Examples

- "Run SELECT * FROM leads LIMIT 10"
- "Create a table for contacts"
- "Show all tables in the database"
