# Web Scraper

Extract content from web pages using HTTP fetch.

## Usage

Provide a URL and choose what to extract: text content, links, headings, or raw HTML.

## Examples

- "Scrape https://example.com for links"
- "Get all headings from this page"
- "Extract text from https://news.site.com/article"
